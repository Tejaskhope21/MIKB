# 🏥 About MIKB

MIKB is a modern hospital equipment selling platform built for hospitals, clinics, and healthcare professionals. The platform simplifies the process of browsing, managing, and purchasing medical equipment online.

It provides:

- Secure Authentication
- Product & Inventory Management
- Order Management
- Cloudinary Image Uploads
- Role-Based Access Control
- Responsive UI for all devices
